# Test Automation Project

## Overview
This project includes:
- Healthcare Claims Testing with Playwright
- E-commerce Automation Testing with Playwright
- API Testing with Postman

## Structure
- `healthcare/` - Claim submission tests
- `ecommerce/` - Online shopping flow tests
- `api/` - Login and endpoint tests

## How to Run

```bash
npm install
npx playwright install
npm run test
npm run api-test
```
